# Snowman-Meltdown

An interactive hangman game for the terminal, developed in Python.


## Features
* Random word selection from a curated list
* Seven incorrect attempts in each round
* Each incorrect attempt is visualized by a snowman that slowly melts
* Error-tolerant input (upper/lower case letters are ignored)
* Validation of user input (only single letters allowed)
* Visualization of letters already used
* Replay Function
